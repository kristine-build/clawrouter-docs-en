## Batch Delete Channels

Requires Administrator Permissions (Admin)

Endpoint

* POST `/`api`/`channel`/`batch\`
* Base URL: https://docs.newapi.pro

Request Content-Type: application/json

Request Body schema (see original docs):

* ids? array

Reference: https://docs.newapi.pro/en/docs/api/management/channel-management/channel-batch-post#request-body

```
curl -X POST "https://docs.newapi.pro/api/channel/batch" \
  -H "Content-Type: application/json" \
  -d '{}'
```

Response

* 200 — Empty
